# Data-Science-Analytics-Interns-Task-

This repository contains a few data science projects focusing on classification, clustering, and dashboards.

## Task 1: Term Deposit Prediction
- Predict whether a bank customer will subscribe to a term deposit.
- Dataset: Bank Marketing (UCI)
- Models: Logistic Regression, Random Forest, explained with SHAP.

## Task 2: Customer Segmentation
- Cluster customers based on spending habits.
- Dataset: Mall Customers
- Method: K-Means, visualized with PCA/t-SNE.

## Task 3: Business Dashboard
- Interactive dashboard to analyze sales, profit, and top customers.
- Dataset: Global Superstore
- Built using Streamlit with filters and KPIs.
